Main Description

The Software Architecture Design generally consists of two work products; the architecture model and the architecture design documentation. The latter can be generated from the architecture model, or be created in another way based on the model.

The Software Architecture Design defines how the software is structured into components and how these interact. The software interfaces are also part of the architecture design.
