Role Descriptor: Software Tester
The Software Tester is responsible for the definition and execution of Software Validation Tests.

Main Description
The Software Tester develops the Software Validation Test Specification based on the Software Requirement Specification and is resposible for validating all Software Requirements. He implements Software Validation Test cases on the testing system (usually a HIL system or equivalent), executes the tests and reports the results.
