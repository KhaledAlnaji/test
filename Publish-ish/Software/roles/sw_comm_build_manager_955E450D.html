Role: Software Build Manager
The Software Build manager maintains the complete software build and ensure all models are properly integrated, configured, baselined and released.

    Main Description
    The Software Build Manager is responsible for the software configuration and configuration management. He integrates all software modules and through final integration- and regressiontesting ensures that all modules operate according to the architectural design. He is the final responsible for each software baseline and ints integrity and takes care of proper configuration of each software module, storage of software, configuration and additional documentation.
    Staffing

    2) Experienced in the Embedded Software domain
    3) Solid understanding of software architecture and interfaces
    4) Experienced in the use of software tooling and targets, such as compilers, linkers, configuration managenent tooling, integration test tools
