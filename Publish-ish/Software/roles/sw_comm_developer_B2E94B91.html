Role: Software Developer
The Software Developer is responsible for software detailed (unit) design and/or software modeling, code development or integration and model, unit and/or back-to-back testing.
Role Sets: All Process Roles, All Process Roles, Development Team, Scrum Team Roles
Relationships
Primary Performs
  • Develop Integration Test Specification
  • Develop Model Test Specification
  • Develop Software Back-to-Back Test Specification
  • Generate Code
  • Perform Formal Review
  • Perform Software Model Test
Additionally Performs
  • Investigate Change Request or Problem Report
  • Maintain Software Configuration
  • Perform Project Quality Audit
  • Perform Software Integration
  • Perform Software Integration Test
  • Perform Software Project Kick-off
  • Raise Software Change Request or Problem Report
Modifies
  • Generated Code
  • Reviewed Work Product
  • Review Report
  • Software Back-to-Back Test Specification
  • Software Integration Test Specification
  • Software Model Test Result
    Main Description
    The Software Developer is the main responsible for the taking software from design to implementation. Based on the output of Software Architecture Design and Software Requirement Specification he creates a detailed design or Software Models and implements or generates the software code. He creates Software Model Test Specifications,  Software Unit Test Specifications and/or Software Back-to-Back Test Specifications and tests the developed models and code before delivering the new software to the Software Build Manager.
