Description
Main Description

An issue list is a list of  running report on a particular problems, its status, and other relevant data. They have a unique reference number which is used to allow the user to quickly locate, add to or communicate the status of the issue.

Besides the reference number and the description of the issue, the issue list will also contain all the attributes of the issue creating a complete overview of the project issues.
