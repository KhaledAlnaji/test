Main Description

The taskboard is used to give an overview of the sprint. Normally the taskboard is split up into three parts; To Do, In Progress and Done. Where all tasks in the sprint backlog are sorted on the taskboard.

In most cases the taskboard is a whiteboard with all tasks written down on post-its, but this does not need to be the case. A large screen with a digitally taskboard can also be used. Key is that there is a place for the team where they can get together to discuss the remaining work and align with each other.
