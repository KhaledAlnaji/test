Main Description

Generated code from the Software Component Model with additional updates to the generated code if these cannot be integrated into the code generation.  This work product is the main outcome of the Generate Code task.

Each function is linked to their corresponding function is the Software Component model to ensure traceability. Later in the development cycle, each function is additionally traced to one or more test cases in the Software Back-to-back Test Specification and to downward documents such as the Software Back-to-back test result specification.
