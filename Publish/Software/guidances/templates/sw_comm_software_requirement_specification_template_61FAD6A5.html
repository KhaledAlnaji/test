Main Description

This is the ReqIF template for the Software Requirement Specification. ReqIF is a standardized format used by many automotive companies and supported by many requirement management tools, such as DOORS, Integrity and Eclipse RMF.

The template can be used under Eclipse RMF-ProR and contains the basic layout and item types needed to create the Software Requirement Specification. To use it, simply store the attached .reqif file under your project, change the filename to the desired format and open the document in Eclipse ProR.
