Term Definition: Epic
Main Description
An Epic is a large type of User Story that will (at some point) be broken down into several user stories. Epics are typically used to describe high level user stories of which the contents are not yet clear enough to be detailed into smaller user stories. They can be seen as "place holders" on the Product Backlog that are waiting for the more detailed requirements become clear.
Also, an Epic can be used in a hierarchical way to bundle a group of user stories that belong together under it.
