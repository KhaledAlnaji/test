Main Description

The Software Back-to-back test Speficitation is the document that holds all tests that must be performed on the on the generated code. This specification is the main outcome of the Develop Software Back-to-back test task.

 Each test is written as a single entity, each with individual parameters such as preconditions and stimuli and is traced to the Software Component Model.
