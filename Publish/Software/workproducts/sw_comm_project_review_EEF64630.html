Description
Main DescriptionRegardless of the size or complexity of the project, it is important to measure the success of a particular project against well-defined criteria. The criteria defined in the Software development plan help to determine whether the project meet its target objectives. The review report provides valuable information to the project sponsor and/or steering committee to assist them in making informed decisions It captures the lessons learnt which will benefit the organisation in future projects.
